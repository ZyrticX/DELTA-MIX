# Legacy Streamlit Application

זו המערכת הישנה מבוססת Streamlit. היא נשמרה כאן למטרות היסטוריות.

המערכת החדשה היא **DeltaMix 2.0** שנמצאת ב-`prediction_engine/` ו-`frontend/`.

## קבצים

- `deltamix.py` - אפליקציית Streamlit הראשית
- `pages/` - עמודי Streamlit
- `utils.py` - פונקציות עזר ל-Streamlit
- `daily_update.py` - עדכון יומי ישן

## שימוש

אם אתה רוצה להריץ את המערכת הישנה:

```bash
streamlit run deltamix.py
```

**הערה:** המערכת החדשה (DeltaMix 2.0) היא המומלצת לשימוש.

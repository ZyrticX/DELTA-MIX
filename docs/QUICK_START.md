## התקנה מהירה

### Windows

1. **הורד והתקן Python**
   - גש ל-https://www.python.org/downloads/
   - הורד Python 3.11 (או גרסה עדכנית יותר)
   - בהתקנה, **סמן את התיבה "Add Python to PATH"**

2. **פתח Command Prompt**
   - לחץ `Win + R`
   - הקלד `cmd` ולחץ Enter

3. **התקן חבילות**
   ```bash
   pip install yfinance pandas numpy openpyxl streamlit plotly scipy
   ```

4. **הורד את הקבצים**
   - צור תיקייה חדשה (למשל `C:\CorrelationSystem\`)
   - העתק את כל הקבצים לתיקייה:
     - `deltamix.py`
     - `correlation_engine.py`
     - `data_fetcher.py`
     - `daily_update.py`

### Mac/Linux

1. **פתח Terminal**

2. **התקן Python (אם אין)**
   ```bash
   # Mac (עם Homebrew)
   brew install python3
   
   # Ubuntu/Debian
   sudo apt-get install python3 python3-pip
   ```

3. **התקן חבילות**
   ```bash
   pip3 install yfinance pandas numpy openpyxl streamlit plotly scipy
   ```

4. **הורד את הקבצים**
   ```bash
   mkdir ~/CorrelationSystem
   cd ~/CorrelationSystem
   # העתק את הקבצים לכאן
   ```

## הגדרת עדכון אוטומטי

### Windows - Task Scheduler

#### שלב 1: צור קובץ BAT
צור קובץ `run_daily_update.bat` בתיקיית המערכת:

```batch
@echo off
cd C:\CorrelationSystem\
python daily_update.py >> daily_update.log 2>&1
```

#### שלב 2: הגדר Task Scheduler
1. פתח **Task Scheduler** (חפש בתפריט Start)
2. לחץ **Create Basic Task**
3. שם: "Correlation Daily Update"
4. Trigger: **Daily**
5. שעה: **23:00** (11 PM - אחרי סגירת השוק האמריקאי)
6. Action: **Start a program**
   - Program: `C:\CorrelationSystem\run_daily_update.bat`
7. Finish!

#### שלב 3: בדיקה
- לחץ ימני על ה-Task
- בחר **Run**
- בדוק את `daily_update.log`

### Mac/Linux - Cron

#### שלב 1: ערוך Crontab
```bash
crontab -e
```

#### שלב 2: הוסף שורה
```bash
0 23 * * * cd /path/to/CorrelationSystem && python3 daily_update.py >> daily_update.log 2>&1
```

פירוש:
- `0 23 * * *` = כל יום בשעה 23:00
- `>> daily_update.log` = שמור פלט לקובץ log

#### שלב 3: שמור ויצא
- לחץ `Ctrl + X`
- לחץ `Y`
- לחץ `Enter`

#### שלב 4: בדיקה
```bash
# רשימת ה-cron jobs
crontab -l

# הפעלה ידנית לבדיקה
cd /path/to/CorrelationSystem
python3 daily_update.py
```

## פתרון בעיות

### בעיה: "streamlit: command not found"

**פתרון:**
```bash
# Windows
python -m streamlit run deltamix.py

# Mac/Linux
python3 -m streamlit run deltamix.py
```

### בעיה: "HTTPError: 403 Forbidden"

**פתרון:** בעיית רשת/חסימת Yahoo Finance
```bash
# נסה להשתמש ב-VPN
# או המתן מספר דקות ונסה שוב
```

### בעיה: הממשק תקוע על "מריץ ניתוח..."

**פתרון:** התחל עם פחות מניות
```
1. שנה "מספר מניות" ל-20
2. נסה שוב
3. אם עובד, הגדל בהדרגה
```

### בעיה: "Memory Error"

**פתרון:** הקטן את כמות הנתונים
```
1. הגדל את "תאריך התחלה" ל-2020 או יותר
2. הקטן "מספר מניות" ל-100
```

### בעיה: התוצאות שונות מהאקסל

**אפשרויות:**
1. **תאריכים שונים** - ודא שהתאריכים זהים
2. **מניות חסרות** - ייתכן שמניה הוסרה מה-S&P 500
3. **עדכון נתונים** - Yahoo Finance מעדכן נתונים, האקסל סטטי

**בדיקה:**
```bash
python test_system.py
```
הסקריפט ישווה את הפרמטרים עם האקסל המקורי.

### בעיה: העדכון האוטומטי לא רץ

**Windows - בדיקה:**
1. פתח Task Scheduler
2. מצא את ה-Task
3. לחץ ימני → **Properties**
4. לך ל-**History** - בדוק שגיאות

**Mac/Linux - בדיקה:**
```bash
# בדוק את ה-log
tail -f daily_update.log

# בדוק את syslog
grep CRON /var/log/syslog
```

### בעיה: הקאש תופס יותר מדי מקום

**פתרון:**
```bash
# במסך הממשק, לחץ "🗑️ נקה קאש"
# או ידנית:
```

**Windows:**
```bash
del /Q data_cache\*
```

**Mac/Linux:**
```bash
rm -rf data_cache/*
```
